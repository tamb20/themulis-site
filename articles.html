How to Add a New Article | طريقة إضافة مقال جديد

الطريقة المبسطة

  1. انسخ أحد ملفات المقالات داخل مجلد articles.
  2. غيّر اسم الملف، مثل: executive-stability.html.
  3. عدّل العنوان والمحتوى داخل الملف.
  4. أضف بطاقة جديدة في صفحة articles.html تشير إلى رابط المقال الجديد.

Recommended File Structure

/articles/article-name.html

Each article has its own standalone page. Add a matching card to the main Articles page and, if relevant, to the Knowledge Center inside the TB Adaptive Model page.
