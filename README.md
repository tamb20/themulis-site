# themulis.com - Bilingual Final Version

Upload all files and folders to GitHub/Vercel project root.

Language switch:
- Click AR to show Arabic RTL version
- Click EN to show English LTR version
- Preference is saved in localStorage


## Adding new articles

1. Open the folder `/articles`.
2. Copy `_article-template.html`.
3. Rename it, for example: `executive-stability.html`.
4. Edit the title, category, date, and article body.
5. Open `articles.html` and add a new article card linking to the new file.
6. If the article is related to TB Adaptive Model™, also add its link in the Knowledge Center section inside `tb-model.html`.

Example link:
`articles/executive-stability.html`
