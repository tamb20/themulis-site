# themulis.com - Final Static Website

This package contains a static bilingual landing page for **Dr. Tariq Mohana Bahwini**.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/portrait.png`

## How to upload to GitHub
1. Create a new GitHub repository.
2. Upload all files and folders exactly as they are.
3. Commit the files.
4. If you want to publish with GitHub Pages:
   - Go to **Settings** > **Pages**
   - Under **Build and deployment**, select **Deploy from a branch**
   - Choose the `main` branch and `/root`
   - Save
5. Your website will be published automatically.

## Notes
- Arabic text is configured to prefer the `Jannat` font if available on the device.
- If you want guaranteed Jannat rendering on every device, add the official font files and connect them with `@font-face` in `styles.css`.
